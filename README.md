# Git Fork & Pull Request 실습 리포지토리

이 리포지토리는 GitHub의 Fork, 수정, Push, Pull Request 실습을 위한 저장소입니다.

## ✏️ 실습 내용

1. 이 리포지토리를 **Fork**하세요.
2. `hello.txt` 파일을 열고 본인의 이름과 간단한 인사말을 추가하세요.
3. 변경 사항을 커밋하고 본인의 GitHub 저장소로 **Push**하세요.
4. 이후 **Pull Request**를 생성하여 제출하세요.
